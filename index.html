Unicode

Универсальная кодировка

История

  • 1988 — первый черновик
  • 1991 — опубликована версия 1.0.0

OpenType

Расширяемый шрифтовой формат

Преимущества

  • кроссплатформенный
  • поддерживает Unicode, 65 000+ символов
  • эффективное сжатие
  • улучшенная типографика (лигатуры, дроби, исторические символы) с автозаменой
  • улучшенный кернинг
  • расширяемый формат 🔥

Форматы

  • EOT — поддерживается только в IE
  • OTF — на основе PostScript
  • TTF — на основе TrueType
  • WOFF — сжатый OpenType шрифт
  • WOFF2 — более лучше сжатый OpenType шрифт

Кастомные шрифты

Пример кода

@font-face {
  font-family: MyFont;
  src: local('MyFont'),
    url('path/MyFont.woff2') format('woff2'),
    url('path/MyFont.woff') format('woff');
}
body {
  font-family: MyFont, Helvetica, Arial, sans-serif;
}
